# discord-new-world-forum-bot
discord bot integration for new world forum

## Installation
* clone repo
* create discord application
    * create discord bot
    * receive token
* create .env
```dotenv
DISCORD_TOKEN=<bot_token>
GUILD_ID=<guild_id>
CLIENT_ID=<bot_id>
```
* save .env
```bash
npm install
npm run dev
```
